## Core Features:

- Raffle Creation: Allow users to create raffles with title, description, image, price per number, total numbers, and draw date.
- Shareable Raffle Link: Generate a shareable link for each raffle with Open Graph meta tags for a visually appealing preview on WhatsApp, including a custom URL slug, image, title, and description.
- Number Grid: Display available, reserved, and paid numbers in a grid format.
- Purchase Flow: Enable users to select numbers, enter their name and WhatsApp number, and display the total amount and payment information (Pix key or QR code).
- Payment Status: Show "Payment pending" message after purchase, reserve numbers, and update status to "Paid" after confirmation, blocking the numbers.
- Raffle Updates: Provide options to join a WhatsApp group and receive raffle updates; include an exportable participant list with name and WhatsApp number.
- Admin Dashboard: Provide a password-protected admin dashboard to create raffles, set details (name, description, image, price, numbers, date, WhatsApp group link), view the participant list, manually confirm payments, and close the raffle.

## Style Guidelines:

- Primary color: Soft yellow (#FFECB3) for CTAs and highlights, providing a warm and friendly feel.
- Background color: Off-white (#F5F5F5) to maintain a clean and simple look.
- Accent colors: Dark gray (#424242) for text, green (#4CAF50) for available/success, and light gray (#BDBDBD) for unavailable states.
- Body and headline font: 'Poppins' for a legible, contemporary feel.
- Use simple icons to represent raffle states and actions, ensuring clarity and ease of use.
- Employ a mobile-first design with rounded cards, large buttons, and a well-highlighted number grid for an intuitive user experience.
- Subtle animations to confirm payment statuses and reflect number selections
